# 지식마루

2022 개정 교육과정 **정보교과 임용시험** 준비를 위한 학습·퀴즈 플랫폼입니다. 별도 설치나 빌드 없이 브라우저에서 바로 열어 사용할 수 있는 정적 웹 페이지로 구성되어 있습니다.

## 구성 내용

- 중학교 정보
- 고등학교 정보
- 고등학교 인공지능 기초
- 데이터과학
- 소프트웨어 생활

## 주요 기능

### 공부 탭 (`study.html`)
- 상단에서 **중학교 / 고등학교** 학교급을 선택합니다.
- 고등학교를 선택하면 정보·인공지능 기초·데이터과학·소프트웨어 생활 4개 과목이 순서대로 이어서 표시됩니다.
- 하위 탭으로 성격·목표 → 내용체계 → 성취기준 → 교수·학습 → 평가 순서에 따라 2022 개정 교육과정 원문 내용을 확인할 수 있습니다.
- 원본 교육과정 PDF·한글 파일 링크도 함께 제공합니다.

### 퀴즈 탭 (`quiz.html`)
- 공부 탭과 동일하게 학교급 탭과 5개 하위 탭(성격·목표/내용체계/성취기준/교수·학습/평가)으로 구성됩니다.
- 교육과정 전체 내용을 바탕으로 만든 **빈칸 채우기 문제 574개**를 과목·영역별로 풀어볼 수 있습니다.
- 정답을 입력해 확인하면 오답일 경우 모범 답안이 함께 표시됩니다.

## 파일 구조

```
2022-study/
├── index.html, study.html, quiz.html   # file/ 하위 실제 페이지로 리다이렉트
└── file/
    ├── index.html                      # 랜딩 페이지
    ├── study.html                      # 공부 탭 페이지
    ├── quiz.html                       # 퀴즈 탭 페이지
    ├── curriculum-data.js              # 교육과정 원문 데이터 (성격/목표/내용체계/성취기준/교수학습/평가)
    ├── quiz-data.js                    # 빈칸 채우기 퀴즈 데이터
    ├── script.js                       # 탭 전환, 렌더링, 정답 채점 로직
    ├── styles.css                      # 스타일
    ├── manifest.json, icon.svg         # PWA 설정
    └── *.pdf, *.docx, *.hwp            # 원본 교육과정 자료
```

## 실행 방법

별도의 빌드 과정 없이 `index.html`을 브라우저로 열면 바로 사용할 수 있습니다. 로컬 서버로 열고 싶다면 예를 들어:

```bash
npx serve .
```

## 기술 스택

순수 HTML/CSS/JavaScript로 작성되었으며, 별도 프레임워크나 빌드 도구를 사용하지 않습니다.
